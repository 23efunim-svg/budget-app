# Семейный бюджет — PWA

Личное финансовое приложение. Работает на Android / iPhone / Windows одинаково.

**Live:** https://23efunim-svg.github.io/budget-app/

## Что умеет

- План vs Факт с прогресс-барами по категориям
- Редактирование сумм прямо с телефона, автосохранение в браузере
- Чекбоксы оплачено/не оплачено
- Копилки с целями и долги с трекингом
- Импорт CSV из Zenmoney — обновляет факт за текущий месяц одним файлом
- Экспорт/импорт JSON для переноса между устройствами
- Работает офлайн после первого запуска (service worker)
- Ставится как приложение на рабочий стол

## Структура

| Файл | Назначение |
|---|---|
| `index.html` | Главный файл, открывается двойным кликом. Весь JSX заинлайнен — один файл работает без сервера. |
| `app.jsx` | Тот же код отдельным файлом для удобства чтения/редактирования. **В `index.html` используется встроенная копия** — при изменении `app.jsx` нужно пересобрать `index.html`. |
| `manifest.webmanifest` | PWA-манифест (иконка, имя, цвета) |
| `sw.js` | Service worker для офлайн-режима |
| `icon-192.png` / `icon-512.png` | Иконки приложения |
| `.gitignore` | Исключает `private/` — папку с личными данными |
| `private/personal-data.json` | **НЕ в git.** Реальные суммы/имена/долги для импорта в приложение. |

## Быстрый старт для владельца

1. Открой https://23efunim-svg.github.io/budget-app/ с телефона (Android/iPhone)
2. Меню браузера → «Установить приложение» / «Добавить на главный экран»
3. В приложении: вкладка «Синхро» → «Импорт JSON» → загрузи `private/personal-data.json`
4. Готово — видишь реальный план мая

## Обновление факта из Zenmoney

1. В Zenmoney: меню → Экспорт → CSV
2. В приложении: «Синхро» → «Выбрать CSV файл»
3. Факт по категориям обновится, прогресс-бары поедут

## Разработка

Приложение — один HTML + babel-standalone. Нет сборщика, нет npm install.

Чтобы изменить логику:
1. Правишь `app.jsx`
2. Пересобираешь `index.html` (скрипт сборки сейчас — bash heredoc; см. историю git)
3. `git commit && git push` — GitHub Pages обновится за ~1 минуту

## Связь с проектом `ya/finance/`

Это приложение живёт внутри `ya/finance/budget-app/`, но git-репозиторий у него **свой** (чтобы отдельно пушить на GitHub Pages, не тащя туда личные md с планами долгов и анализом CSV).

Цикл:
- Zenmoney CSV → `ya/finance/00-context/zen_YYYY-MM-DD.csv` → ежемесячный разбор Claude Code в VS Code (обновляет план долгов, бюджет)
- Новый план → обновление `private/personal-data.json` → импорт в приложение
- В течение месяца: правки в приложении на телефоне (факт, копилки)
- Раз в неделю: экспорт JSON из приложения → `ya/finance/00-context/budget-state-*.json` → сверка план/факт в VS Code
